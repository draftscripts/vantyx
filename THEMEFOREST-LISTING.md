Cover (`cover.png`, 2340×1560) + `preview-1…9.png` go in the **description/gallery** (already embedded via CDN), not the Files section.

**Theme Preview = a ZIP of preview images** (NOT an HTML demo). Per the form: "ZIP file of images (png/jpg)". The first image appears on the item page + search results and must be **590×300**, named `01_<name>.png`; additional images are named `02_<name>.png`, `03_…` in the order you want them shown (they may be cropped to the 590×300 aspect in search).
