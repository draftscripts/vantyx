# VantyX — listing assets

Preview images for the **VantyX** admin dashboard template, hosted for use in the
ThemeForest item description via CDN.

## Images (`previews/`)

| File | Size | Use |
|------|------|-----|
| `cover.png` | 2340×1560 | Cover / inline preview |
| `thumbnail-590x300.png` | 590×300 | Listing thumbnail |
| `preview-1.png` … `preview-9.png` | 2100×1400 | Feature preview screens |

## CDN usage

Embed in HTML (jsDelivr, cached + fast):

```html
<img src="https://cdn.jsdelivr.net/gh/draftscripts/vantyx@main/previews/cover.png" width="590" alt="VantyX">
```

Raw GitHub alternative:

```
https://raw.githubusercontent.com/draftscripts/vantyx/main/previews/cover.png
```
